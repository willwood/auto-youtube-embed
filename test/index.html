Auto YouTube Embed

The Auto YouTube Embed uses Javascript to take a video ID, URL or embed code, from an HTML data attribute. It extracts the video ID and automatically converts it into a mobile-optimised (responsive) embedded YouTube video.

Video ID example:

<div data-yt-embed="6axSB3T4vbk"></div>

Video URL example:

<div data-yt-embed="https://www.youtube.com/watch?v=6axSB3T4vbk"></div>

Shortened URL example:

<div data-yt-embed="https://youtu.be/6axSB3T4vbk"></div>

No-Cookie example:

<div data-yt-embed="https://www.youtube-nocookie.com/embed/6axSB3T4vbk"></div>

YouTube Embed Code example:

<div data-yt-embed='<iframe src="https://www.youtube.com/embed/6axSB3T4vbk" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen></iframe>'></div>

If the possibility exists of YouTube embed code getting used, like in this example, enclosing the data-yt-embed attribute value inside single quote marks ( ' & ' ) resolves HTML validation issues and correctly parses the markup.

YouTube Shorts example:

<div data-yt-embed="https://youtu.be/PA3S88aa4ZU" class="yt_short"></div>

Add a class of yt_short to the <div> container, to force a 9/16 aspect ratio.
